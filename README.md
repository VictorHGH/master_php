# Master PHP

Repositorio de practicas y apuntes de curso para PHP, organizado por modulos.

## Estructura

- `01_repaso_html/`
- `02_introduccion_php/`
- `index.php` (entrada principal)

## Requisitos

- PHP 8.x
- Servidor local (por ejemplo `~/Documents/programacion/php`)

## Uso

Con el stack Docker activo, acceder a:

- `http://127.0.0.1:8082/master_php/`

Tambien puedes ejecutar scripts por terminal segun el modulo:

```bash
php index.php
```

## Nota

Este repositorio esta orientado a aprendizaje incremental y ejercicios de referencia.
